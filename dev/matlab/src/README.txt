Ptolemy II consists of a group of Java packages that support
heterogeneous concurrent modeling and design.

The Ptolemy II documentation can be found in doc/index.htm 

The Ptolemy II website is at
http://ptolemy.eecs.berkeley.edu/ptolemyII

To build Ptolemy for ProActive:

0) you must have Matlab installed on your computer

1) Define the PTMATLAB_DIR environment variable to the root directory of your Matlab.

2) Define PTII to ProActive_Path/dev/matlab/src

3) run configure

3) run make

you'll find ptolemy.jar libptmatlab.so (under linux) and ptmatlab.dll (under Windows) in the ProActive/dev/matlab dir
