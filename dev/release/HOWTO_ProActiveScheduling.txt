
For a major or minor release (aka x.y.0):
	- Create a dedicated branch
		- Set serialVersionUid to xy for all serializable class
		- Check copyrights
	- Build the release
		- cd SCHEDULER_HOME
		- dev/release/release-[rm|all] . x.y.z $JAVA_HOME
		- release can now be found in /tmp/ProActiveScheduling-x.y.z(.tar.gz)


NB : Do not forget XSD schema export for major release !
