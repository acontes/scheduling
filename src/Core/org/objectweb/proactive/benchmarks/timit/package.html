Provides the main classes for creating timers and events. The TimIt class that contains the main method that parses arguments and launches the class specified in the configuration file. Multiple examples are provided in the org.objectweb.proactive.examples.timit package.

The util package contains timers and events related classes.

The pics package contains all charts generation related classes.

The config package contains all the xml parsing related classes that you can easily extend with you own tags in order to use them in the configuration file.

For more information about timers and the concept behind TimIt, see the documentation
